# hammer-shared-lib

Shared utilities for HammerSpade and StarMetal.

## Features
- Shared item name/lore formatting with MiniMessage color support
- Shared resource pack request/delivery manager

## Build and install locally

```bash
mvn install
```

## Use in another plugin

```xml
<dependency>
  <groupId>io.github.aleksireede</groupId>
  <artifactId>hammer-shared-lib</artifactId>
  <version>1.0.0-SNAPSHOT</version>
</dependency>
```
